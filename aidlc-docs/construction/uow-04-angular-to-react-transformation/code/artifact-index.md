## Package-Level Notes

- `@spa-bridge/transform-angular-react` is a provider-neutral transformation package that emits React-oriented drafts rather than final React source files.
- The package depends on `@spa-bridge/core-model` for shared contracts and on `@spa-bridge/source-angular` for analyzed Angular source evidence.
- The built-in rule set includes an ecosystem-aware finalize rule that turns known Angular 15 / NgRx / third-party library usage into explicit manual-review and provider-neutral handoff artifacts.
- The workspace root build and test scripts now include the new package.
